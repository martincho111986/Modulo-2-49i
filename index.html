Crear un repositorio en github

  1. Ingresa a la página de github
  2. ¿Tienes cuenta?
  3. NO: Crear cuenta
  4. Iniciar sesión
  5. Click new respository
  6. ¿Desea crear repositorio privado?
  7. SI: Check privado
  8. ¿Agrega mas complementos?
  9. Si: Check complementos
  10. Crear repositorio

Clonar un proyecto

  1. Ir al repositorio
  2. Hacer click en code
  3. Copiar la dirección <url>
  4. En la consola escriba git clone <url>
  5. Presiona Enter
  6. ¿Desea crear una rama?
  7. SI:
    1. ¿Desea crear desde vs?
    2. SI: git brancha <nombre de la rama>; No: crear desde github
  8. git checkout <nombre rama>
  9. git add .
  10. git commit -m "descripcion"
  11. Verificar que estoy en la rama correcta
  12. git push
